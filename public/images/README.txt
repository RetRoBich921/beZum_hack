Для создания рекламного изображения (reklama1.png) необходимо:

1. Создать изображение размером 300x200 пикселей
2. Использовать яркие цвета
3. Добавить текст "РЕКЛАМА" крупным шрифтом
4. Сохранить в формате PNG
5. Поместить файл в эту директорию с именем reklama1.png

Рекомендуемые цвета:
- Фон: #FFD700 (золотой)
- Текст: #FF0000 (красный)
- Акценты: #0000FF (синий)

Шрифт: Arial Black или Impact
